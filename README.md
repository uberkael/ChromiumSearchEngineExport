# Chromium Search Engines Export/Import
Tool for **Export/Import** search engines between **Chromium**-based **browsers**.

## Use
Using UV or Poetry:
```bash
uv run main.py
# or
poetry run python main.py
```
Using requirements:
```bash
pip install -r requirements.txt
python main.py
```
